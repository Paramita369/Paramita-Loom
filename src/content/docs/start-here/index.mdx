Paramita Loom is arranged around reading modes rather than platform features.

If you are new here, use this order:

1. Read the homepage to understand the editorial stance.
2. Visit [Knowledge](/knowledge/) for stable topic maps and stitched references.
3. Visit [Guides](/guides/) if you want procedure and application.
4. Visit [Notes](/notes/) for lighter, still-useful fragments.
5. Visit [Reviews](/review/) when you need judgments, tradeoffs, and buying context.
