# 記憶體價格飆高，Raspberry Pi調漲4GB以上機型價格

## 一句話版本

Raspberry Pi 因為 LPDDR4 記憶體一年內貴咗 7 倍，被迫再加價 4GB 以上機型，呢件事重要係因為連平價單板電腦都頂唔順，代表上游記憶體壓力已經直接打到我哋會買嘅邊緣運算硬件。

## 點解重要

- **成本結構變咗**
- 4GB、8GB、16GB 機型分別加 25、50、100 美元，唔係小修小補，而係記憶體容量越高就越傷，表示之後做設備選型時，RAM 已經唔可以再當「順手加多啲」嘅便宜選項。
- **唔係單一產品出事**
- 今次唔只 Raspberry Pi 4/5，加價仲擴散到 Raspberry Pi 500、Compute Module 4/5、開發套件同 AI HAT+ 2，意味住成條高記憶體產品線都受壓，唔係換另一個 Pi 型號就自然避得開。
- **官方已經連續第三次調價**
- 自 2025 年底以來因記憶體成本連續三次改價，重點唔係今次加幾多，而係呢種波動開始變成持續狀態，採購同報價如果仲用舊假設，之後會一路失真。
- **低價替代方案反映官方策略轉向**
- 新出 3GB 版 Raspberry Pi 4，售價 83.75 美元，說明官方而家唔係硬食成本，而係主動把產品切層，鼓勵用家按實際需要買較低記憶體配置，減少為「預留空間」埋單。
- **低容量機型暫時安全，但原因係供應鏈唔同**
- 1GB、2GB 同 Raspberry Pi 400 冇加價，主要因為用 LPDDR2、庫存又夠，呢點提醒我哋要睇清楚零件世代同 BOM 結構，唔好只睇產品名以為同系列就受同一種風險。
- **這不是 Raspberry Pi 自己的問題**
- LPDDR4 供應鏈本身同手機、平板、輕薄筆電共用，當 AI 資料中心同高價應用搶產能時，Pi 呢類低毛利產品自然排後面，所以之後類似 SBC、邊緣裝置、模組化硬件都可能同步受影響。
- **官方講明屬暫時性，但短期內仍要接受波動**
- Eben Upton 提到若零件價格回落會再降價，代表供應商同品牌方都未視這次調整為永久定價重設，但對我哋而言，短中期規劃仍然要把價格上下震盪當成常態，而唔係例外。

## 我哋點睇

- 如果我哋有用 Raspberry Pi 做本地控制器、測試台或者邊緣節點，之後規格要先由實際 workload 倒推，避免習慣性選 8GB 或更高版本。
- 做採購預算時，應該把「同型號但不同記憶體版本」拆開估，而唔好再用單一 Pi 單價做成本模型。
- 對需要大量部署嘅場景，3GB 或低容量版本可能會變成更實際嘅標準配置，前提係先驗證會唔會影響穩定性、快取空間同本地模型/服務負載。
- 如果 roadmap 依賴 Compute Module 或 AI HAT 類配件，最好及早重算 BOM，同時預留供應波動空間，避免到落單先發現整套方案成本已經變樣。
