# Exclusive: Runway launches $10M fund, Builders program to support early-stage AI startups

## 一句話版本

Runway 唔再只係做 AI 影片模型，佢而家用一個 1,000 萬美元基金加埋 Builders 計劃，想提早綁定一批初創去試同擴大佢嘅「video intelligence」生態。

## 點解重要

- **由工具公司走向平台公司**：Runway 唔係單純賣模型能力，而係開始主動影響「邊啲公司會喺佢上面長出嚟」，即係話佢想由供應商升級做生態位中樞，對合作方同競爭對手都會有更大牽引力。
- **投資方向透露產品路線**：佢將 thesis 分成基礎 AI 架構、foundation model 上層應用、同新媒體創作分發，反映 Runway 認為價值唔只喺模型本身，而係由 infra 去到應用層都仲有大把位可以卡。
- **「video intelligence」係佢想講嘅下一個故事**：文章重點唔係影片生成靚唔靚，而係視像會變成更通用嘅理解同互動介面，意味 Runway 想從創意工具跨去更多產業場景。
- **Builders 計劃其實係低成本市場研發**：免費 API credits 加開放 Characters 等能力，表面係扶持初創，實際上都係用外部團隊幫佢測需求、找 use case、驗證產品邊界。
- **提供到 Series C，都幾進取**：唔只係 pre-seed 或 seed，連較後期團隊都可以食到 credits，代表 Runway 想滲透成長中公司，而唔係只做早期品牌曝光。
- **最高 50 萬美元支票唔算大，但夠早期卡位**：呢個金額未必足以主導一輪融資，但足夠令 Runway 以戰略投資者身份提早入場，換取技術採用、合作優先權，甚至產品方向影響力。
- **佢已經唔係由零開始試**：過去一年半已經低調投咗幾間公司，說明呢次公布更似正式化同擴張，而唔係臨時起意。
- **被投公司類型值得留意**：由 LanceDB 呢類 AI infra，到 Tamarind Bio 呢類生命科學應用，都顯示 Runway 唔係只睇媒體內容，而係睇多模態能力點樣滲去其他垂直領域。
- **多模態已經由口號變成資本配置**：文中明講下一代模型要建基於影片、音訊、圖像、文字一齊運作，呢個判斷而家已經唔止體現在研究敘事，仲直接反映到投資同 BD 行動。
- **同業模式開始收斂**：OpenAI、Perplexity、CoreWeave 都搞基金，代表基礎模型公司開始用「投資 + credits + 平台 access」去搶早期創業者，之後生態競爭會更似雲平台戰，而唔只係模型 benchmark 戰。
- **Runway 規模唔大，但想放大外部槓桿**：公司只有約 150 人，自己無可能追晒所有場景，所以用投資同計劃去外包探索，呢種打法可以令細團隊都食到更大市場面。
- **Characters 反映產品正走向即時互動代理**：由單次生成內容，轉向有樣有聲、可即時互動嘅 video agent，表示 Runway 想搶嘅唔只係內容製作，而係下一代人機介面入口。
- **對創業團隊嚟講，資金同算力開始綁埋一齊**：拎 Runway 錢或者 credits，唔單止係財務支持，仲可能代表技術棧傾向會被預設，之後轉平台成本會變高。
- **對市場嚟講，應用層機會仍然大**：Runway 公開講佢自己無法親自做晒所有 use case，等於間接承認模型之上仲有大量未被吃透嘅應用空間。

## 我哋點睇

- 如果我哋做產品規劃，呢單新聞提醒我哋要留意「模型公司平台化」風險，唔好只評估 API 能力，仲要評估會唔會被對方投資、生態政策、credits 結構鎖死。
- 如果我哋考慮接入生成式視像或 agent 能力，應該優先問清楚供應商嘅平台意圖，因為一旦對方想做生態主場，合作條件、價格、分發權重都可能之後再變。
- 對 OpenClaw 呢類控制面導向系統嚟講，最值得學唔係 Runway 投基金，而係佢用外部 builders 幫自己發掘場景；我哋如果日後開擴充點，都應該保持「控制面係真相來源，外部能力只係受控插件」呢條線，避免生態反過來推翻核心控制邏輯。
