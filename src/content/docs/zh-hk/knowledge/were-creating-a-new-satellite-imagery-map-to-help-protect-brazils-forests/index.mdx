# We’re creating a new satellite imagery map to help protect Brazil’s forests.

## 一句話版本

Google 同巴西政府整咗一張高細節歷史衛星地圖，令森林砍伐可以被更準確定位同量度，重點唔止係資料更靚，而係監管同問責終於有更硬淨嘅底圖。

## 點解重要

- **量度基準升級**
- 巴西早年曾面對破紀錄砍伐，呢張地圖提供咗當時國土景觀嘅高精度快照，等之後講「有冇改善」時唔再只靠粗略估算，而係有較可信嘅對照基線。
- **監管可以由大概變成具體**
- 新影像可以見到更細緻嘅森林斑塊，意味住執法部門唔係只知道某區有問題，而係更接近知道問題發生喺邊一塊、嚴重到咩程度，行動優先次序會更實際。
- **精度提升直接影響決策質素**
- Google 話新資料最高比舊有影像精細六倍，呢個唔係單純技術炫耀；對政策、巡查、保育資源分配嚟講，細節多一截，誤判同漏判空間就細一截。
- **技術處理解決咗歷史影像難題**
- 佢哋處理咗成千上萬張歷史衛星圖，做咗去雲同色彩校正，反映真正價值唔只係「有圖」，而係將原本難直接用嘅資料整理成可比較、可判讀、可落地使用嘅資產。
- **公共可用性令影響唔止於政府內部**
- 呢批資料已放入 Google Earth 同 Earth Engine，代表唔係封閉式專案；研究者、公民團體、媒體同其他機構都可以用同一套底圖睇問題，公共討論會更有共同事實基礎。
- **環境議題被轉成可驗證進度**
- 森林保護成日卡喺口號層面，但有咗更準確嘅地理證據之後，政府講自己做成點、外界質疑成效，都可以落到具體位置同變化上，透明度會高好多。
- **顯示大型平台公司喺公共治理入面嘅角色**
- 呢件事唔只係地圖更新，而係科技公司直接幫政府建立監測能力；對外部團隊嚟講，呢種合作模式值得留意，因為平台能力可以變成治理基礎設施一部分。

## 我哋點睇

- 如果我哋做任何同營運、風險監測、審核相關嘅系統，呢單案例提醒我哋：先建立可信基線，再談改善，否則之後所有 KPI 都會好虛。
- 對產品設計嚟講，最有啟發唔係「高解析度」本身，而係將原始資料處理到執法人員真係用得落手；我哋都應該優先做可行動、可核對嘅輸出，而唔係只堆資料。
- 如果之後我哋要向用戶或管理者展示成效，應該學佢呢種做法：同時提供公開可見嘅視圖同可分析嘅底層資料，咁先容易建立信任。
