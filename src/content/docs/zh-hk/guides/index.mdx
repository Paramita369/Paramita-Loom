指南是反思與應用之間的橋。它應該有結構、可測試、可重複，但仍保有這個站的語氣與細緻度。

其中〈如何在 Mac 安裝 OpenClaw〉同〈如何在 Windows 安裝 OpenClaw〉而家講的，都是普通用家用官方 CLI 進入公開預覽的主線；Windows 版現時推薦 `WSL2 + Ubuntu` 路線。

〈如何確認 OpenClaw 已正常啟動〉而家已收成普通用家裝完之後的啟動檢查頁，重點係 gateway、dashboard 同 doctor。
