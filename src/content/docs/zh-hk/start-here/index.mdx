Paramita Loom 以閱讀模式來編排，而不是以功能或系統模組來分類。

如果你是第一次進來，建議這樣看：

1. 先讀首頁，理解這個站的編輯立場。
2. 進入 [知識](/zh-hk/knowledge/) 看較穩定的主題圖譜與參照頁。
3. 進入 [指南](/zh-hk/guides/) 看可操作的流程與步驟。
4. 進入 [筆記](/zh-hk/notes/) 看較輕量、仍保有現場感的內容。
5. 需要判斷與取捨時，再進入 [評測](/zh-hk/review/)。
